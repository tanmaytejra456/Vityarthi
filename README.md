 Project title:- PROPERTY ANALYSER
 
 Overview of the project:-
 The Property analyzer  is a pure Python desktop application built using the Tkinter GUI framework. It focuses on the calculation of the property valuation using per square feet value , and also generate rental agreement  and provides Broker details.

 
 Features:- 
 1. Land value calculation
 2. Broker Detals
 3. Rental agreement generator

Technology/tools used:-
1. Python 3.x
2. GUI framework
   
Steps to Install & Run the Project:
1. Install "Python"
2. Save the file: Property analyser.py
3. Run the script

Instruction for testing:
1. Calculation Error Handling: Enter non-numeric text (e.g., "abc") into the Land Area field and click "Calculate Value." An "Error: Please enter valid numerical inputs." message should appear.
2. Agreement Copy Test: Generate an agreement and click "Copy Text to Clipboard." Paste the contents into a text editor to confirm the full agreement text was copied.
